# 実験プロジェクト

このプロジェクトは、入力されたURLを所定のエンドポイントに `?url=` クエリで渡し、返ってきたHTMLをブラウザ上でレンダリングする簡易ツールです。

## 使い方
- 画面の入力欄に取得したいページのURLを入力し、「コンテンツの取得」ボタンを押します。
- アプリは、設定された「所定URL」（エンドポイント）に `?url=<入力URL>` を付与してアクセスし、レスポンスHTMLを画面に表示します。
- 表示はサンドボックス化された iframe 上で行われ、レスポンス内のスクリプトは実行されません。

## エンドポイントの設定方法（必須）
以下のいずれかの方法でエンドポイント（所定URL）を指定してください。

1. ページURLにクエリで指定する
	- 例: `index.html?endpoint=https://example.com/proxy`

2. グローバル変数で指定する
	- ページ読み込み前に次のように設定します。
   
	```html
	<script>
	  window.ENDPOINT_BASE = 'https://example.com/proxy';
	</script>
	<script src="main.js"></script>
	```

どちらの場合も、実際のリクエストは `https://example.com/proxy?url=<入力URL>` のように送信されます。

## 注意事項
- クロスオリジンの取得には、エンドポイント側でCORSを適切に許可している必要があります。
- セキュリティ上、取得HTMLはサンドボックス iframe で表示され、スクリプトは実行されません。
- デザインは最小限です。必要に応じて `style.css` を調整してください。
